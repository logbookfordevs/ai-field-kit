# AI Field Kit Page

A React + Vite landing page for [AI Field Kit](https://github.com/logbookfordevs/ai-field-kit), the portable rules, skills, workflows, and MCP setup for multi-agent developers.

## Quick Start

```bash
pnpm --dir apps/site install
pnpm site:dev
```

The local app runs with Vite. The default development URL is:

```text
http://127.0.0.1:5173
```

## Build

```bash
pnpm site:build
```

## Stack

- React
- Vite
- Tailwind CSS v4
- SVGL shadcn registry icons
