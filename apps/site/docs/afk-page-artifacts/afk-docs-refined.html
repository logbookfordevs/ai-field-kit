AI Field Kit

Curated patterns for autonomous coding.

AFK is not a framework. It is a set of resolution rules that shape how your coding agents behave, what skills they possess, and how they perceive your local project standards.

Core Logic

The Resolution Engine.

AFK resolves setup from an explicit source, a saved default source, or the built-in AFK defaults. Local manifest files are materialized setup inputs, not the primary authoring surface.

1
Explicit Sourceafk setup --source
2
Saved Sourceafk setup --default-source
3
Built-in Sourcelogbookfordevs/ai-field-kit

Source Rule

The active setup source wins. Use afk show to inspect what AFK will read, and use afk show --local only when you need to inspect materialized local manifest files.

Source Model

Shape your setup source.

A setup source is a repository or directory with convention-compatible manifests for rules, skills, MCPs, utilities, hooks, and presets.

{
  "rules": [
    {
      "id": "react-patterns",
      "target": ["clauderc", "gemini"],
      "content": "Always use functional components..."
    }
  ]
}
{
  "skills": [
    "grill-me",
    "afk-to-prd-spec",
    "afk-execution-tracking"
  ]
}
{
  "mcps": [
    {
      "id": "stitch-mcp",
      "args": ["--port", "63162"]
    }
  ]
}

Keep rules atomic

Don't create massive rules. Keep them focused: testing-standards.md, error-handling.md. Atomic rules are easier to mix and match across project types.

Command Surface.

The AFK CLI is your primary tool for previewing setup, resolving targets, and inspecting the source AFK will use.

afk setup

Full interactive setup for rules, skills, MCPs, utilities, and hooks

afk show

Inspect the active setup source

afk setup rules --dry-run

Preview what files will be modified before committing

afk setup --default-source your-org/dev-kit

Save a setup source for later dry-runs and scripted setup

Installation

Machine Setup.

Install the CLI, preview the setup, then apply only the areas and detected targets you want.

Install the CLI

curl -sSL afk.run | bash

Choose your profile

AFK will ask if you want Minimal, Standard, or Full. Standard is the right choice for most developers.

Verify the environment

afk info
