# Resume Turbo

Resume from repo artifacts, not chat memory.

Find the prior goal package, handoff docs, plan, facts, board state, and branch context.

If the prior `goal.md` is done, stale, or scoped away from the remaining work, do not restart that goal. Use the artifacts as Plannotator input for a new focused goal package.

If a deferred seam is under-specified, run a short grilling pass before `plannotator-setup-goal`.

Then continue the normal Turbo route from the fresh goal package.
