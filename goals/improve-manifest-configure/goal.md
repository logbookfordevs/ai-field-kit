# Improve AFK Manifest Configure

Turn `afk manifests configure` into a powerful interactive manifest editor. Users should be able to update AFK manifests from the command line by loading existing manifest files, editing rules, skills, MCPs, utils, and hooks, adding/removing/toggling entries, previewing changes, and confirming writes without manually opening JSON files.

Use `facts.md` as the shared understanding of required behavior.

Use `plan.md` as the approved execution plan.

Done when the accepted facts are implemented, automated verification is added for the facts marked for automation, and the validation commands in `plan.md` pass.
