# AFK Skill Profiles

Implement AFK skill profiles under `afk skills profiles ...` so users can enable one or more named skill groups and temporarily disable unrelated global shared skills to reduce active skill/context noise.

Use [facts.md](./facts.md) as the shared behavior contract and [plan.md](./plan.md) as the approved implementation plan.

Done means the profile catalog, runtime state, CLI commands, safety behavior, local/global profile management, docs, and automated tests are implemented and validated with the AFK package checks.
